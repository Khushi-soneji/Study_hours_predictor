# 📚 Study Hours Predictor

A Machine Learning project that predicts the optimal study time based on a student's learning conditions and generates a personalized study plan with productivity recommendations.

## ✨ Features

- Predicts personalized study hours using Machine Learning
- Compares multiple regression models to select the best performer
- Evaluates models using MAE, RMSE, R² Score, and Cross Validation
- Generates a personalized Pomodoro study schedule
- Provides smart study suggestions based on user inputs
- Calculates study effort level and explains prediction results

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📊 Machine Learning Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regressor (SVR)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Khushi-soneji/Study_hours_predictor.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the notebook:

```bash
Study_Time_Predictor.ipynb
```
